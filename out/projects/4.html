Digital accessibility audit website

Project Description

This ongoing web application is designed to audit websites for digital accessibility issues based on WCAG standards. Built with Node.js and React, it allows users to submit URLs for evaluation, view accessibility reports, and receive recommendations for improvements. The platform promotes inclusive design by highlighting key areas such as contrast issues, missing ARIA labels, and keyboard navigation errors.

Technologies

Node.jsReactJavaScript

Features

  • Fully accessible layout following WCAG guidelines.
  • Homepage presenting the agency’s mission and services.
  • Sections on digital accessibility and its impact.
  • Responsive design and contact form for inquiries.
